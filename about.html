About Us

This project is inspired by the terrible music made today. Sometimes I want to go back to 2011 and listen to the top songs from that year. And with this web application it will allow me to listen to past songs with ease as if I were on spotify. Billboard has a way to see the top 100 songs but doesnt have a way to play the song directly from the website. So with this application we had in mind. We would be able to get the top 100 songs of each month and stream them directly from the source.
