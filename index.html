Top 100 Songs of Each Month

Topic Overview

My idea for this project is to create a web application that provides the top 100 songs in an album for each month since 1990. It will draw information from the top 100 billboards API based on the month you chose. It will then generate an album for you to listen to based on that months top 100 songs. Within the application you are given the options to:

  • Skip
  • Select specific songs
  • Select specific timings in the song
  • See how many listens it has

    • This project is inspired by the terrible music made today. Sometimes I want to go back to 2011 and listen to the top songs from that year. And with this web application it will allow me to listen to past songs with ease as if I were on spotify. Billboard has a way to see the top 100 songs but doesnt have a way to play the song directly from the website. So with this application I have in mind. We would be able to get the top 100 songs of each month and play them directly from the source.

    Potential Applications

    • Spotify Type App
      • This would be an app where you look up the date and can listen to music from the past. This can be usefull for people looking for songs that they might remember and to get a blast from the past.
    • Guessing Game App
      • I think a guessing game based on the past top 100 charts would be fun. It can keep score and will give you a random month and year with options of songs to chose from.
